# StockPulse 📈

A stock market dashboard project with real-time insights.

## 🚀 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Spring Boot (Java)
- Database: MySQL

## ⚙️ Setup Instructions

### 1. Clone repo
git clone https://github.com/your-username/StockPulse.git

### 2. Setup Database
- Open MySQL
- Create database: stockpulse
- Import stockpulse.sql

### 3. Run Backend
- Open in Eclipse
- Run Spring Boot Application

### 4. Run Frontend
- Open in VS Code
- Use Live Server
